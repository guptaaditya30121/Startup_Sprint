## Setup

Create Virtual Environment

```bash
  virtualenv venv
```

Activate Virtual Enviroment

```bash
  source venv/bin/activate
```

Install Packages

```bash
  pip install -r "requirements.txt"
```

Run the Server

```bash
  python manage.py runserver
```
